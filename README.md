📝 Blog na Azure Cloud

📌 Descrição

Este projeto consiste no desenvolvimento e deploy de uma aplicação web de blog utilizando serviços da Microsoft Azure.

O objetivo principal é demonstrar, de forma prática, como aplicações web podem ser hospedadas na nuvem, explorando conceitos como escalabilidade, alta disponibilidade e facilidade de gerenciamento.

A aplicação permite a criação e visualização de conteúdos em um ambiente acessível via navegador, evidenciando o uso da computação em nuvem no contexto real.

☁️ Tecnologias e Serviços Utilizados

Microsoft Azure

Azure App Service (ou Azure Container Apps)

HTML

PowerShell

VS Code

🎯 Objetivos do Projeto

Desenvolver uma aplicação web funcional (blog)

Realizar o deploy na nuvem utilizando Azure

Compreender os conceitos fundamentais de Cloud Computing

Explorar serviços de hospedagem de aplicações web

Demonstrar escalabilidade e gerenciamento em ambiente cloud

🏗️ Arquitetura do Projeto

A arquitetura do sistema é simples e focada em demonstrar o fluxo básico de deploy em cloud:

🔹 Aplicação Web (Frontend)

Desenvolvida em HTML

Responsável pela interface do usuário

Permite visualizar (e possivelmente criar) posts

🔹 Camada de Hospedagem (Cloud)

Aplicação publicada no Azure App Service ou Azure Container Apps

Responsável por manter a aplicação disponível online

Gerencia escalabilidade e disponibilidade automaticamente

🔹 Deploy e Gerenciamento

Utilização de PowerShell para automação de comandos

Publicação e gerenciamento dos recursos na Azure

🚀 Funcionalidades

Criação de posts (dependendo da implementação)

Visualização de conteúdos no navegador

Interface simples e acessível

Aplicação hospedada na nuvem

📊 Resultados

Com este projeto foi possível:

Aplicar conceitos de computação em nuvem na prática

Realizar o deploy de uma aplicação web na Azure

Compreender o funcionamento de serviços como App Service e Container Apps

Demonstrar como aplicações podem ser hospedadas de forma escalável e gerenciável

📎Conclusão

O desenvolvimento desta aplicação foi uma experiência prática completa. Desde a configuração do ambiente até o deploy final na Azure, cada etapa serviu para consolidar meu entendimento sobre como sistemas modernos operam. O maior aprendizado foi perceber que uma aplicação web de sucesso não depende apenas de um bom código, mas de uma base sólida na nuvem que suporte seu crescimento.
