📝 Blog na Azure Cloud
📌 Descrição

Este projeto consiste na criação de um blog hospedado na nuvem, utilizando serviços da Microsoft Azure, como App Service ou Container Apps.

O objetivo é demonstrar, na prática, como aplicações web podem ser desenvolvidas, implantadas (deploy) e gerenciadas na nuvem, aproveitando recursos como escalabilidade, alta disponibilidade e facilidade de manutenção.

☁️ Tecnologias e Serviços Utilizados

Microsoft Azure

Azure App Service 

VSCode

HTML

PowerShell

🎯 Objetivos do Projeto

Criar uma aplicação web de blog funcional

Realizar o deploy da aplicação na nuvem

Explorar serviços da Microsoft Azure

Compreender conceitos de Cloud Computing

Demonstrar como hospedar aplicações web de forma escalável

🚀 Funcionalidades

Criação e visualização de posts

Interface web acessível via navegador

Hospedagem em ambiente cloud

📊 Resultados

Com este projeto, foi possível:

Aplicar conceitos de computação em nuvem na prática

Realizar o deploy de uma aplicação web na Azure

Compreender como funciona a hospedagem de aplicações em cloud

Explorar recursos como escalabilidade e gerenciamento de serviços
